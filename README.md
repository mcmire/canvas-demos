
## Vector math

### Dot product and perp dot product

Dot product:      v1.mag * v2.mag * cos(theta)
                  v1.x*v2.x + v1.y*v2.y
Perp dot product: v1.mag * v2.mag * sin(theta)
                  -v1.y*v2.x + v1.x*v2.y
